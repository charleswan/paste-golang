# 1. 为什么要学习设计模式

```
思考一个问题, 为什么要学习设计模式呢
当你设计并开发好一个新服务的时候, 你的心态是: 
心情舒服, 代码流畅, 清爽美丽
当来第一个需求的时候, 你的心态是: 
好啊
当来了 几个 需求之后, 你的心态是: 
呃。。。
。
。
。
如此往复。。。
一年之后。。。

噩梦
所以, 我们要学习设计模式
```
